# Clip assets

The default demo uses frame-level replay data instead of shipping large video files. To use real clips, place short MP4 files in this directory and point `replay_path` at the clip path when building the edge runtime with OpenCV enabled.

Recommended clip naming:

- `quiet_porch.mp4`
- `person_approach.mp4`
- `person_linger.mp4`
- `package_delivery.mp4`
- `package_removed.mp4`
